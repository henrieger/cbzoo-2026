# Congresso Brasileiro de Zoologia - 2026

This repository contains the scripts, datasets and source code for the poster presented by Henrique L. Rieger and advised by Gustavo A. Ballen at the 2026 edition of the Congresso Brasileiro de Zoologia.

The directory `poster` has all the source code and assets for the poster, while the `analysis` directory contains the scripts and datasets used in the multiple analyses.
